Figma 插件推荐二

2020-06-21 2 min read # 设计 # Figma # 效率

🌟代表我常用的插件 ❤️代表跺脚推荐

**🎈Color 插件** 1.Brands Colors 品牌颜色集合，包括 Facebook、Twitter、Instagram 等在内的650多个品牌的主色

😋 Mockup 类插件
1.Layout Grid Visualizer 布局网格可视化工具（用于导出展示布局）🌟
2.Font Replacer 字体替换器，轻松替换字族和字体样式

📝 文本插件
1.Font Replacer 字体替换器，轻松替换字族和字体样式
2.Font Scale 按可选规则生成和谐的字体规范（快速生成字体规范）🌟

😋 基础插件
1.Themer 快速创建和替换样式主题（从发布的样式库里替换，也就是要付费版才可以）🌟
2.Instance Swap 快速搜索替换实例或插入实例🌟

🖋 矢量图形处理插件
1.LottieFiles 导入Lottie动画，支持SVG、GIF导出🌟
2.Isometric 快速制作等距轴测图 🌟
3.Fill Rule Editor 编辑矢量文字的填充规则（显示正确布尔运算）

💠 布局&网格类型
1.AutoGrid Figma 自动网格，自定义行、列、间距 ❤️
2.Layout Grid Visualizer 布局网格可视化工具（用于导出布局展示）🌟
3.Frame Resizer 从锚点调整 frame 大小，而无需调整子图层的大小和更改约束，常用尺寸可以存为预设 🌟

注：部分插件参考 Figma 群内聊天记录
想学习 Figma 的朋友可以去Figmacn 学习，网站里许多 Figma 视频教程和知识值得学习。
插件推荐会攒到一定数量发布如果想实时更新可以去我的 Notion 文档里看完整版和更新记录。
